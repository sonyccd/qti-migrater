About QTI Migrator

QTI Migrator

A powerful command-line tool for migrating QTI (Question and Test Interoperability) content between different platforms and formats.

Version: 1.0.0

Author: QTI Migration Team

License: MIT

What is QTI?

QTI (Question and Test Interoperability) is an international standard that enables the exchange of assessment content and results between different systems. QTI Migrator helps you convert and migrate your QTI content seamlessly.

Cross-platform CLI tool written in Go

Core Capabilities

  • Parse and validate QTI content
  • Convert between QTI versions (1.2, 2.0, 2.1, 2.2)
  • Batch processing of multiple files
  • Command-line interface for automation
  • Cross-platform support (Windows, macOS, Linux)
  • Progress tracking and detailed logging

Input/Output Formats

  • QTI XML (all versions)
  • Common Cartridge
  • SCORM packages
  • CSV export for question banks
  • JSON format for web applications
  • Custom format templates

Advanced Features

  • Content validation and error reporting
  • Media file handling and migration
  • Metadata preservation
  • Custom transformation rules
  • API for programmatic access
  • Plugin system for extensions
Installation Guide

Installation

Download Binary

Download the latest release for your platform:

Install via Go

go install github.com/yourusername/qti-migrator@latest

Build from Source

git clone https://github.com/yourusername/qti-migrator.git cd qti-migrator go build -o qti-migrator ./cmd/main.go

Usage Examples

# Convert QTI 1.2 to QTI 2.1 qti-migrator convert --input quiz.xml --output quiz_v21.xml --from qti12 --to qti21 # Validate QTI content qti-migrator validate assessment.xml # Batch convert all files in directory qti-migrator batch --input-dir ./assessments --output-dir ./converted --to qti22
Requires Go 1.19 or later
